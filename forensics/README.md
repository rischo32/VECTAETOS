# VECTAETOS — Epistemic Forensics Layer

Status: Observational Layer  
Agency: None  
Execution Power: None  

---

## Purpose

This layer provides structural observation of the epistemic field Φ.

It does not:

- detect errors  
- fix problems  
- optimize trajectories  

It describes:

- local tensions  
- structural gradients  
- epistemic density  

---

## Architecture

Φ → Vortex → multi_run.json  
↓  
Forensics (feature extraction)  
↓  
Reader (descriptive projection)  
↓  
Human interpretation  

---

## Constraints

- no decision logic  
- no optimization  
- no intervention  

Forensics observes structure.  
It does not act upon it.
