# VECTAETOS / VECTLAB CORE — MVP DRIFT GUARD

## Status

This file is a semantic guard for the minimal deterministic MVP.

## Correct reading

```text
VECTLAB core = deterministic audit CLI
AI layer = later optional LLM Adapter / Research mode
Φ = untouchable field, not runtime state
ledger = provenance, not truth
hash = integrity, not ontological verdict
h_topo = observable, not score
```

## Forbidden reading

```text
VECTLAB core = AI agent
Φ = mutable runtime state
ledger = truth source
hash = proof of ontological validity
h_topo = safety score
EK = control layer
audit = execution authority
CLI = decision system
```

## Boundary

This MVP may compute external audit observables.

It must not:

- compute truth,
- compute K(Φ),
- compute κ,
- select trajectories,
- control a Vortex,
- mutate Φ,
- write back into Φ,
- claim deployment safety,
- claim empirical validity.

## Canonical sentence

This core is a deterministic audit CLI, not an AI agent.
