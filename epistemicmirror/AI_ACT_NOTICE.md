## AI ACT NOTICE

Epistemic Mirror — Web Interface
Status: Informational Transparency Statement
Version: 1.0
Scope: Public Web Deployment Only

1. Purpose of This Notice
This document provides a transparency statement regarding the Epistemic Mirror web interface in relation to Regulation (EU) 2024/1689 (Artificial Intelligence Act, “EU AI Act”).
This document is not legal advice.
It is a technical and structural clarification of the system’s operational characteristics.

2. Nature of the Web System
Epistemic Mirror is a deterministic web-based projection interface.
The system:
Accepts textual input from a user
Computes structural scores (Width, Height, Depth)
Classifies structural deformation type
Selects a reflective prompt from a fixed dataset using deterministic hashing
The system does not:
Learn from user data
Adapt after deployment
Update internal parameters
Store personal user histories
Perform profiling
Perform optimization
Execute autonomous decision-making
Rank, score, or classify individuals

3. Deterministic Operation
All outputs are generated via:
Rule-based structural scoring
Static prompt datasets
Deterministic index selection
Identical structural inputs may produce identical outputs.
The system does not perform probabilistic inference, predictive modeling, or behavioral analysis.

4. Output Character
The output of Epistemic Mirror consists exclusively of:
Reflective questions
Structural projection summaries
The system does not generate:
Recommendations
Advice
Decisions
Predictions
Behavioral instructions
Risk assessments
Outputs are informational reflections only.

5. User Responsibility
All decisions taken by a user remain exclusively the responsibility of the user.
Epistemic Mirror:
Does not assume agency
Does not exercise authority
Does not claim correctness
Does not provide professional guidance
The system operates as a projection mechanism, not as an advisory or decision-making tool.

6. Data Handling
Epistemic Mirror:
Does not store user inputs on the server
Does not build user profiles
Does not conduct automated decision-making under GDPR Article 22
Does not perform biometric or sensitive data analysis
If analytics tools are introduced in the future, they will be documented separately.

7. Preliminary EU AI Act Position
Based on its architecture:
Epistemic Mirror does not:
Perform adaptive inference
Operate with autonomous objective selection
Automate decisions
Influence physical or digital environments
If interpreted as a generative interface under Title IV transparency provisions, the system satisfies transparency requirements by clearly informing users that outputs are automatically generated and non-authoritative.

8. Scope Limitation
This notice applies only to:
Epistemic Mirror — Web Interface v1.x
Any future introduction of:
Adaptive models
Optimization loops
Decision automation
Learning mechanisms
would require reassessment under Regulation (EU) 2024/1689.
End of document.
