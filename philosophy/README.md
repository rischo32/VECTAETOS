# PHILOSOPHY

## Philosophical and Ontological Layer of VECTAETOS

### Status
Conceptual layer  
Non-formal  
Non-executive  
Non-authoritative  
Upstream of research, but not above formal anchors  

---

## 0. Purpose

The `/philosophy` directory contains the philosophical, ontological, and conceptual material of VECTAETOS.

It is a place for:

- theory,
- reflections,
- conceptual origins,
- meaning structures,
- philosophical notes,
- ontological distinctions,
- pre-formal formulations,
- interpretive essays.

This directory exists to preserve the thinking from which formal structures may later be clarified.

It is not the formal core.

It is not runtime logic.

It is not a decision layer.

It is not an authority over `/formal`.

---

## 1. Relation to `/formal`

The `/formal` directory contains canonical formal definitions.

The `/philosophy` directory may explain, illuminate, motivate, or expand the meaning around those definitions.

However, `/philosophy` must not silently redefine anything already frozen in `/formal`.

If a philosophical text conflicts with a canonical formal anchor, the formal anchor remains authoritative for the framework.

Philosophy may prepare future formalization.

It does not override existing formalization.

---

## 2. Relation to `/research`

The `/research` directory is for investigation, experiments, comparisons, exploratory mechanisms, technical hypotheses, and unfinished research artifacts.

The `/philosophy` directory is for conceptual grounding.

/philosophy = why it means what it means
/research   = how it may be explored
