## 1. Purpose

Tento priečinok slúži ako miesto pre teórie metamatematiky VECTAETOS.

Je určený na:

- formálne vysvetlenia,
- symbolické čistenie,
- mapovanie ekvácií,
- porovnávanie matematických vrstiev,
- skúmanie hraníc reprezentovateľnosti,
- opravy zápisu,
- audit symbolických kolízií,
- rozlíšenie medzi ontológiou, formalizáciou a projekciou.

Tento priečinok nie je:

- Frozen Ontological Core,
- rozhodovací systém,
- runtime mechanizmus,
- optimalizačná vrstva,
- reward funkcia,
- autorita nad Φ,
- miesto na tichú zmenu VECTAETOS 1.x.

Pre opravy LaTeXu, značenia, symbolických kolízií alebo nejasných zápisov.

```text

Ontology change: none
```

### 5.4 COMPARISON NOTE

Pre porovnanie dvoch dokumentov alebo vrstiev.

```text
Status: COMPARISON NOTE
Verdict: descriptive, not authoritative
```

### 5.5 DRAFT

Pre neuzavretý text.

```text

Boundary review required
```

## 10. Research Humility

Metamatematika nesmie predstierať viac, než vie ukotviť.

Kanonická veta:

```text
Radšej priznané nevieme než vyrobená pravda.
```

Rozlišuj:

```text
coherence ≠ proof
notation ≠ ontology
formal elegance ≠ evidence
projection ≠ interpretation
audit ≠ authority
hypothesis ≠ theorem
```

Ak výrok nie je opretý, označ ho ako:

```text
hypothesis
draft
interpretation
notation proposal
boundary question
```
