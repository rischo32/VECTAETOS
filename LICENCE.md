4. File-Level and Artifact-Level Overrides

If a file, directory, Zenodo record, DOI release, paper, PDF, dataset, archive item, or external artifact declares a different license, that explicit license governs that artifact alone.
Historical files preserved under /archive/ retain their declared historical status and do not define the current repository license stack unless explicitly stated.
In particular, historical VNAL materials are preserved for provenance and lineage. New implementation releases should use VNAL-1.1 where explicitly declared.

5. Non-Retroactivity

This license stack does not automatically rewrite:
