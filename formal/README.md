## 6. Relational Structure

The relational structure `R` is antisymmetric:

```math
R_{ij} = -R_{ji}
```

and diagonal-free:

```math
R_{ii} = 0
```

This implies:

```math
R \in \mathfrak{so}(8)
```

The independent relational pairs are:

```math
\binom{8}{2} = 28
```

`R` does not encode:

- dominance,
- authority,
- preference,
- rank,
- command,
- reward,
- decision pressure,
- optimization gradient.

`R` encodes relational tension only.
