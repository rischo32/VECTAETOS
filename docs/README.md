# VECTAETOS™ Interface Prototype

Static interactive SPA prototype inspired by the supplied screenshots.

## Run locally

Open `index.html` directly in a browser, or serve the folder with a local static server. No build step is required.

## Deploy

This is a static site. It can be deployed to GitHub Pages, Netlify, Cloudflare Pages, Vercel, or any static host.

## Included modules

- Boot screen
- Window manager
- Terminal
- Papers
- DOI Registry
- License Stack
- Field Topology canvas
- Notes with local browser persistence
- Settings with local browser persistence
- About VectaetOS

## Drift guard

This is an interface prototype.

- Boot sequence is UI state, not ontology initialization.
- Topology graph is a descriptive projection, not Φ itself.
- Terminal is a local command UI, not an agent.
- DOI registry is metadata display, not truth validation.
- Notes are local workspace state, not ontological memory.
- Settings do not tune Φ.
