## 0. PURPOSE

Tento dokument obsahuje druhú dávku fragmentov pre ZMYSEL korpus.

Fragmenty sú odvodené z ôsmich axiomatických ťažísk VECTAETOS:

- INT — Zámer
- LEX — Existencia
- WIS — Múdrosť
- REL — Vzájomnosť
- VER — Pravda
- LIB — Sloboda
- UNI — Jednota
- CRE — Tvorba

Tento dokument nič nerozhoduje.

+ Neobsahuje parser.
+ Neobsahuje váhy.
+ Neobsahuje skórovanie.
+ Neobsahuje runtime.

Slúži iba ako významová zásoba pre budúcu prácu so ZMYSEL korpusom.

---

## 1. CORPUS PRINCIPLE

+ ZMYSEL nezhromažďuje výroky ako pravidlá.
+ ZMYSEL zhromažďuje fragmenty, ktoré nesú existenciálnu nosnosť.
+ Fragment nie je príkaz.
+ Fragment je stopa významu.

---

## 2. FORMAT

Každý fragment má tvar:

```text
ID je stabilný identifikátor fragmentu.
AXIOMATIC CENTER určuje vzťah k jednému z 8 ťažísk.
FRAGMENT je samotná veta korpusu.
FIELD NOTE je deskriptívna poznámka.
BOUNDARY určuje, čo fragment nesmie znamenať.
```
