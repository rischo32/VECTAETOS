R(i,j)

# 3. Cycle Curvature

Define cycle curvature:

Δ(i,j,k) = R(i,j) + R(j,k) + R(k,i)

This measures the relational inconsistency of a triangle.

Interpretation:

Δ = discrete curvature of the epistemic field.

If the relations in the triangle are perfectly compatible:

Δ(i,j,k) = 0

If relations conflict, curvature emerges.

---

# 4. Geometric Meaning

Cycle curvature represents the failure of relational closure.

In a consistent relational geometry:

R(i,j) + R(j,k) + R(k,i) = 0

This corresponds to a flat relational triangle.

Non-zero curvature indicates a local distortion of the epistemic field.

# 5. Global Field Geometry

The geometry of Φ is determined by the distribution of curvature across all triangles.

Let:

T = set of all triangles in K₈

Then the global curvature distribution is:

{ Δ(t) | t ∈ T }

This distribution characterizes the internal geometry of the field.
