0. Purpose

This directory contains forensic and audit mechanisms that:

- observe epistemic field projections
- localize structural inconsistencies
- expose topological breakdowns

This layer does NOT:

- modify Φ (epistemic field)
- influence K(Φ) or κ
- generate decisions or recommendations
- introduce feedback into the system

Audit exists strictly outside Φ.

«Π_forensic : Φ → 𝒫(Ω)»

- Φ = epistemic field (ontology)
- Ω = set of localized incoherences
- Π_forensic = projection (not interpretation)
